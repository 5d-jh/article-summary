**[Firefox Addons](https://addons.mozilla.org/ko/firefox/addon/mygist/)** (Supports Firefox Android)

**[Chrome Web Store](https://chromewebstore.google.com/detail/mygist/miemlfekdmpbbendbclldhhegahdaifd)**

Summarize webpages privately using your own local LLMs.

Requires LM Studio and have server running.

For Firefox Android, you can run summarization on remote server. Set remote server URL on settings page(e.g. LAN address or Tailnet address). On server side, need to toggle 'Serve on Local Network' ON on LM Studio.
